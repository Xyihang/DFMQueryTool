# DFM查询工具使用说明

## 新增功能

本工具现在支持保存和加载用户配置，无需每次都手动输入认证信息。

## 使用方法

### 1. 配置用户信息
- **OpenID**: 您的QQ OpenID（必填）
- **Access Token**: 访问令牌（可选）

### 2. 保存配置
- 在用户配置区域填入您的信息
- 点击"保存配置"按钮
- 配置将保存到 `user_config.ini` 文件中

### 3. 查询数据
- 选择查询类型
- 点击"查询数据"按钮
- 系统会自动使用保存的配置进行查询

## 配置文件格式

配置文件 `user_config.ini` 格式如下：
```ini
[User]
openid = 332908108AE3BCD62DCE9843072E00FE
access_token = CD7193582B50A3284DEE0B15749FEF23
```

## 安全说明

- 配置文件保存在程序运行目录下
- 请妥善保管您的认证信息
- 不要分享您的配置文件给他人
- 建议定期备份配置文件

## 故障排除

1. **提示"配置不完整"**
   - 检查是否已填写OpenID
   - 点击"保存配置"按钮

2. **查询失败**
   - 确认网络连接正常
   - 检查认证信息是否正确
   - 查看错误信息中的具体原因

3. **配置文件丢失**
   - 重新填写用户配置信息
   - 点击保存配置按钮

## 支持的查询类型

- 每日密码
- 烽火地带收益Top3
- 全面战场数据
- 战场周报数据
- 烽火周报数据
- 特勤处状态
- 货币资产查询
